# MERN Google Auth

Sign In & Sign Up using google oauth and passport.js.

## Run Locally

Clone the project

```bash
  git clone https://github.com/hammadhz/pms-server.git
```

Go to the project directory

```bash
cd pms-server
```

Write cmd to run server Locally

```
 npm start
```

## Tech Stack

**Server:** Node, Express, Mongodb
